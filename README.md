# AdventureForge Splash Page

Marketing landing page for AdventureForge - professional trip planning software for outdoor guides and adventure groups.

## Files

- `index.html` - Main landing page (self-contained, uses Tailwind CDN)
- `adventure_forge_logo.png` - AdventureForge logo

## Deployment

This is a static site that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Simply point the hosting service to this repository and set `index.html` as the entry point.
